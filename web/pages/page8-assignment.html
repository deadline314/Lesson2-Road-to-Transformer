作業 — 設計你自己的模型

Homework Assignment · Due in 2 weeks

「請學弟們下面挑一個來做，當然都做是最好的。」

作業說明

以下三個題目請至少挑一個來完成。每個題目都涵蓋了本次課程的核心概念， 從 RNN 到 Self-Attention 到 BERT，選擇你最有興趣的方向深入實作。

題目一：RNN 文字分類

1
手刻一個簡單的 RNN（可用 PyTorch）
2
跑一個情感分析或簡單翻譯任務
3
感受 sequential processing 的特性與限制

題目二：Mini Self-Attention

1
用 PyTorch 寫一個 single-head attention
2
跑一個視覺化 attention map 的圖片
3
（加分）生成可互動的視覺化工具

其實有現成套件可以用：BertViz — 可以視覺化各層各頭的 attention pattern。

題目三：BERT Fine-tuning

1
用 HuggingFace 拉一個 BERT 模型（可自行選擇版本）
2
Fine-tune 一個下游任務
3
任務選擇：分類任務、NER、問答等皆可

繳交方式

• 程式碼：GitHub repo（含 README 說明如何執行）
• 報告：簡單說明你的設計思路、實驗結果
• 截止日期：兩週後（詳見課程公告）
